兩百字內的個人簡介

大家好，我是陳柔妤，新北三重人，畢業於國立勤益科技大學——應用英語系，目前定居台中，過去曾擔任信義房屋的房屋銷售業務，現為一名保健食品客服。自小對電腦相關資訊相當有興趣，雖然高中及大學都就讀英語系，但仍有持續關注電腦相關資訊。畢業後出社會選擇先進房仲業接受磨練與洗禮，後續則選擇相對靜態且固定時間上下班的上班族工作，好讓我可於平日下班、假日時間好好進修，期間利用各式線上教學平台學習前端程式語言如：HTML、CSS、JavaScript、Vue 等，逐步累積，期待能達成兒時願望——成為一名電腦工程師。

為了成為軟體工程師，曾做過什麼努力？若有具體專案作品請分享給我們

為此，我廣泛採用線上線下各種資源，如：書籍、網路社群、論壇、文章、免費及付費線上教學課程等，並持續利用日常下班時間及週末時光來探索、學習、練習程式語言，前前後後約兩年的時間。雖偶有被工作、生活打斷的時候，但仍努力讓自己浸泡在程式語言之中，多看多聽、學習前輩、神人的思維模式，並努力消化、內化成自己的東西。對於作品的部分，目前除學習課程作業之外，尚未有專屬的個人作品，我期待透過本集訓營的密集訓練，提高自己的實力，做出一份個人作品，並藉此成功轉職。

  • 課程作品——家教網

如果參與這個計畫，會怎麼安排學習時間？

為了全心投入 WeHelp，我抱著破釜沈舟的決心，已向公司提離職，將於七月開始安排學習時間：平日早上九點到中午十二點，下午一點到傍晚六點，晚上八點至晚上十點。假日則視情況按表操課，將自己視為全職考生，全心浸泡於程式學習中，為未來工作做準備。

是否有想要加入的軟體公司？為什麼想加入該公司？

雖然我並非本科生，但我希望能以自身英文的優勢結合程式專業，往外商發展，並以海外駐點，或者全遠端的工作為目標，實現環遊世界、四海為家的工作型態，邊工作邊玩，Work hard, play hard!

請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？

自開始自學程式語言以來，遇過不少挫折，最令我印象深刻的是初次接觸到框架時，當時直播班的上課問答過程，發現自己雖然有提前預習過影音課程，卻無法及時的回答出老師的問題，對於老師的提問，有時會懷疑自己的理解，相較於其他同學的秒答秒回，內心的挫敗感是非常重的，甚至也懷疑自己是不是很笨，不適合寫程式。當發現自己有這樣龐大的負面情緒時，我就會抽空安排半小時到一小時的時間來運動，或出去走路散散心，透過流汗、活動筋骨等方式讓大腦製造多巴胺幫助自己放輕鬆、減輕負面情緒，我也發現這樣能夠提升學習時的專注力以及學習效率，讓自己的學習旅程更加持久、健康。

關於這份申請網頁，請分享一個你開發時的技術心得。

對於這份申請網頁，除了使用 HTML 及 CSS 之外，也有套用 aos 套件，提升整份文件的動態感；想起剛開始學習程式語言時，試著寫幾段文字，就能夠呈現在畫面上的感動與成就感，是隨著時間更迭也不會消逝的悸動，我期待自己能夠藉著課程，增加自己創造感動的能力與實力，能夠靠著自己的雙手以及一台筆電，沒有時差的完成各式各樣的作品，呈現給需要的人，帶他們進入有著無限可能的網路世界。

其他想要對我們說的事情？

回想剛開始接觸程式語言，是某次偶然在 YouTube 上看到彭彭老師的教學資源，我記得是 Python 的基礎課程，那時候 Python 正夯，我邊看著老師教學一邊試著操作看看，雖然只是基礎中的基礎，但也因此感受到原來自己也能夠寫程式，這是開啟我學習程式語言的一個啟蒙，想先在此謝謝彭彭老師引領進門。而後續我也有嘗試透過各種線上教學資源來學習、探索自己的興趣，發現自己想走前端這條道路。隨著時間流逝，自己也習得了一些語言及框架，但對自己的能力還不是很有自信，所以決定報名參加彭彭老師的集訓營，透過嚴厲密集的學習步調推自己一把。這是我第一次申請 WeHelp Bootcamp，為此，我拿出破釜沈舟的決心，已經和現職公司主管提出離職申請，讓自己有足夠的時間能夠全心投入、浸泡在程式語言的世界之中，跟著老師及同學們一起成長，並順利轉職成軟體工程師，我已經迫不及待開始集訓了！
