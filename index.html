WeHelp 4th Application

兩百字內的個人簡介

大家好，我是陳柔妤，新北三重人，畢業於國立勤益科技大學——應用英語系，目前定居台中，過去曾擔任房屋銷售業務，現為一名電商客服。這兩份工作經歷讓我有很高的抗壓性、臨場反應及表達能力，對於團隊合作與跨部門溝通也非常熟悉；也因為這兩份背景截然不同的經歷，讓我更確定自己想要的是高專業性、能夠長期累積、目標在於解決問題的職涯發展，故決定重拾兒時夢想，並利用下班、假日時間學習，逐步累積，朝軟體工程師邁進。

為了成為軟體工程師，曾做過什麼努力？若有具體專案作品請分享給我們

為此，我廣泛採用線上線下各種資源，如：實體書籍、網路社群、論壇、文章、免費及付費線上教學課程等，並持續利用日常下班時間及週末時光來探索、學習、練習程式語言，前前後後約一年半的時間。雖偶有被工作、生活打斷的時候，但仍努力讓自己浸泡在程式語言之中，多看多聽多接觸、學習前輩、大神的思維模式，並努力消化、內化成自己的東西。對於作品的部分，目前除學習課程作業之外，尚未有專屬的個人作品，我期待透過本集訓營的密集訓練，逼迫自己、提高自己的實力，做出一份個人作品，並成功轉職成軟體工程師。

如果參與這個計畫，會怎麼安排學習時間？

為全心投入 WeHelp 的密集課程，我抱著破釜沈舟的決心，已向公司提離職，六月初就會先自行再次複習之前學過的線上課程及資源，提升學習手感與基礎，七月開課後則依照以下學習時間：平日早上九點到中午十二點，下午一點到傍晚六點，晚上八點至晚上十點。假日則視情況按表操課，維持每週至少 50 小時的學習份量，將自己視為全職考生，全心浸泡於程式學習中，讓自己可以和本科生比較，甚至脫穎而出，也為未來工作做準備，非轉職成功不可！

是否有想要加入的軟體公司？為什麼想加入該公司？

因為大學就讀應用英語系，我希望能以自身英文的優勢結合程式專業，往外商發展，故只要有外商公司的機會，我都有非常高的願意加入，未來也會以海外駐點，或者全遠端的工作為目標，實現環遊世界、四海為家的工作型態，邊工作邊玩，Work hard, play hard !

請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？

自開始自學程式語言以來，遇過不少挫折，最令我印象深刻的是框架的學習。在遇到框架直播班的上課問答時，我發現自己雖然有提前預習過影音課程，卻無法及時的回答出老師的問題，對於老師的提問，有時會懷疑自己的理解，相較於其他同學的秒答秒回，內心的挫敗感是非常重的，甚至也懷疑自己是不是很笨，不適合寫程式。當發現自己有這樣龐大的負面情緒時，我就會抽空安排半小時到一小時的時間來運動，或出去走路散散心，透過流汗、活動筋骨、接觸大自然等方式讓大腦製造多巴胺幫助自己放輕鬆、減輕負面情緒，我也發現這樣能夠提升學習時的專注力以及效率，讓自己的學習旅程更加持久、健康、正面。

關於這份申請網頁，請分享一個你開發時的技術心得。

對於這份申請網頁，除了使用一般的 HTML 及 CSS 之外，也有套用 aos 套件，提升整份文件的動態感，讓其不要看起來太書面；弄著弄著覺得滑到底部之後要往上滑有點麻煩，於是想到可以加入頂置按鈕，方便閱覽標題美景，於是爬文完成這一份簡易的申請文件，也想起剛開始學習程式語言時，試著寫幾段文字，就能夠呈現在畫面上的感動與成就感，是隨著時間更迭也不會消逝的悸動。我期待自己能夠藉著 WeHelp 的訓練，提升自己的程式實力，提高創造感動的能力，能夠靠著自己的雙手以及一台筆電，沒有時差的完成各式各樣的作品，呈現給需要的人，帶他們進入有著無限可能的網路世界！

其他想要對我們說的事情？

回想剛開始接觸程式語言，是某次偶然在 YouTube 上看到彭彭老師的教學資源，我記得是 Python 的基礎課程，那時候 Python 正夯，我邊看著老師教學一邊試著操作看看，雖然只是基礎中的基礎，但也因此感受到原來自己也能夠寫程式，這是開啟我學習程式語言的一個啟蒙，想先在此謝謝彭彭老師引領進門。而後續我也有嘗試透過各種線上教學資源來學習、探索自己的興趣，發現自己想走前端這條道路。隨著時間流逝，自己也習得了一些語言及框架，但對自己的能力還不是很有自信，所以決定報名參加彭彭老師的集訓營，透過嚴厲密集的學習步調推自己一把。這是我第一次申請 WeHelp Bootcamp，為此，我已拿出破釜沈舟的決心離職，讓自己有足夠的時間能夠全心投入、浸泡在程式語言的世界之中，跟著老師及同學們一起成長，並順利轉職成軟體工程師，我已經迫不及待開始集訓了！
